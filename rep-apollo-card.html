Two short jobs in Apollo. One takes 5 minutes and you never touch it again. The other takes 60 seconds each morning. That is the whole manual.

Already handled for you: the 4 custom fields are created across the whole workspace. You are not creating fields. You are just picking them from a menu and dropping them into your own sequence. No typing field names, ever.

The 4 fields you'll be inserting

These already live in your Apollo. You'll pick each one from the { } variable menu. They carry the custom copy Claude writes for every contact.

1
Open your sequence and plug in the Day-1 email
~3 min
  1. Left menu → Sequences → click your sequence name.
  2. Click the Day 1 email step to open the editor.
  3. Click inside the Subject box. Click the { } button. Search Gather_Email_Subject and click it. The subject now shows one blue chip. ✓
  4. Click inside the Body. Delete anything already there. Click { } again, search Gather_Email_Body, click it. Body should show only one blue chip. ✓
  5. Click Save.
One rule that matters: always insert from the { } menu. Never hand-type the field name. Typing it breaks the merge and sends a blank email.
2
Drop the script + LinkedIn note into the other steps
~2 min
  1. Open each Call step and the LinkedIn video step. In the notes box, click { } and insert Gather_Script. If a step's note has no { } button, skip it — the script shows on the contact card during your task anyway.
  2. Open the LinkedIn connection step. Click { } and insert Gather_Connection_Note.
  3. Leave the sequence INACTIVE. You're wired for good. ✓
Each morning Part 2 — Import and send. 60 seconds. Claude already did the research, sourcing, enrichment, and writing while you slept.
1
Import the CSV, then activate
~60 sec

Claude Slacks you a staged batch and a file called apollo-import.csv. This step drops the custom copy onto each contact.

  1. In Apollo, top right → Import → CSV.
  2. Upload apollo-import.csv from the Slack message or your run folder.
  3. When asked how to find existing contacts, match on Email.
  4. Map the four Gather_ columns to the matching fields. Names are identical, so they usually auto-match.
  5. Click Import.
  6. Open one contact → Preview the Day-1 email → confirm you see real copy, not an empty {{ }}. ✓
  7. Review the batch → click Activate. Sending starts.
Never activate before the import runs. An un-imported contact has empty fields, and Apollo will send a blank email. Order is always: Import → Verify → Activate.
